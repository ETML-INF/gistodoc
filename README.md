# Projectil - Project Time Lapse

Une application qui exporte les issues Github dans des documents

L'outil nécessite une authentification par [Personal Access Token](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens) (PAT) classique pour Github, qui doit avoir le scope 'repo' et 'read:project'.
